libdnn
======

C++ Library of Deep Neural Network

## Prerequisite
You need to install NVIDIA CUDA toolkit (at least 5.0)
(Here'e the link https://developer.nvidia.com/cuda-toolkit)

## How to Install ?
Simply type
```
./install-sh
```

## How to Use ?
There're 3 programs, dnn-init, dnn-train, dnn-predict.
